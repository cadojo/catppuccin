# Catppuccin Format

## Installing

```bash
quarto use template cadojo/catppuccin
```

This will install the extension and create an example `.qmd` file that you can use as a starting place for your work.

## Using

This extension bundles several theme files under `_extensions/catppuccin`.
These are extensions of the built-in Quarto themes. It's recommended to
simply reference these files in your Quarto frontmatter.

- [latte](_extensions/catppuccin/latte.scss)
- [frappe](_extensions/catppuccin/frappe.scss)
- [macchiato](_extensions/catppuccin/macchiato.scss)
- [mocha](_extensions/catppuccin/mocha.scss)
- [oled](_extensions/catppuccin/oled.scss)

## Format Options

There are no format options for this extension.
